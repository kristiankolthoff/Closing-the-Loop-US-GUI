## Data Availability

The dataset used in this project is **not hosted in this repository**.

Instead, it is publicly available through the official **KIT RADAR platform**, as the data is under the rights and management of the **Karlsruhe Institute of Technology (KIT)**.

Access the dataset here:  
https://radar.kit.edu/radar/de/dataset/441ukf65ekhvzqzh

### Notes
- The KIT RADAR platform is a research data repository maintained by KIT for publishing and managing datasets.
- Access conditions, licensing, and usage restrictions are defined by KIT and the dataset authors.
- Users are encouraged to download the data directly from the official source to ensure integrity and proper citation.

### Citation
If you use this dataset, please cite it according to the citation guidelines provided on the KIT RADAR page.
